# 福州大学《嵌入式系统综合设计》实践（2023年12月）

本次实践课程包含了一系列针对嵌入式系统开发的综合实验，旨在通过实践学习深入理解嵌入式系统的设计、开发与应用。实验内容覆盖了开发环境搭建、云平台开发、多媒体处理、图像处理技术、视频编解码等多个领域。
